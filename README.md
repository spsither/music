# Collection of Tibetan songs from a YouTube playlist

## Make sure dependencies are installed
```sh
pip install yt_dlp
sudo apt update
sudo apt install ffmpeg
```

## Run the `main.py` script

## Transfer to phone

1. Install VLC on your iPhone
Go to App Store → Search: VLC for Mobile → Install.

2. Connect iPhone and Ubuntu to the same Wi-Fi

3. Open VLC → Tap Browse (bottom) → Enable “Sharing via WiFi”
    It shows a local address like:
    http://192.168.1.25:8080/

4. On Ubuntu, open a browser and go to that address

5. Drag and drop your .mp3 files onto the page
